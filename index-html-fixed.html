About ZOARCH Lab Inventory

This application was created to manage and visualize the Zoological-Archaeological Lab Inventory database. It provides a user-friendly interface to browse, search, update, and analyze the specimen collection.

Features
  • Interactive dashboard with key metrics
  • Comprehensive inventory browser with filtering and search
  • Statistical visualizations of the collection
  • Data import and export functionality
  • Add and edit inventory entries
  • Browser-based storage for offline capability
How to Use

This application uses your browser's local storage to maintain data between sessions. To get started:

  1. Import your Excel data using the Update Data section
  2. Browse your inventory in the Inventory section
  3. View statistics and visualizations in the Statistics section
  4. Add or edit entries as needed
  5. Export your updated data when finished
Technologies Used
  • HTML5, CSS3, and JavaScript
  • Bootstrap 5 for responsive design
  • Chart.js for data visualization
  • DataTables for enhanced table functionality
  • SheetJS for Excel file handling
  • LocalStorage API for data persistence
